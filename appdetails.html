Get To Know About The App

It is usually hard for us to keep track of the things we have to do in a particular day. We usually end up doing things that we have not planned to do and end up leaving out what is very important.

It is on this note that this app was create to make life a bit more orderly. With this App, the user is able to create their own account and also login into an existing account.

The user is also able to create a To-Do List and add items to the list. The user is also be able to create more lists and also delete the un-wanted lists.
